Welcome to the Utah Digital Media Arts Festival Unofficial Webpage

The DMAF is a yearly festival showcasing the best work of Digital Media students from across the state of Utah, across many different categories.

CATEGORIES

Swag Categories

Sticker Design
Create a sticker to promote the festival.
All festival participants will receive a sticker with the winning design
T-Shirt Design
Create a single-color t-shirt design to promote the festival.
All festival attendees receive the winning t-shirt

Digital Video Production

Narrative
Plan, film, and edit a video that tells a story
Non-Fiction
Plan, film, and edit a video that informs or educates. This may include public service announcements, news stories, broadcast highlights, commercials, etc.

Website

Web Design
Create a well designed website

Animation

Stop Motion
Use stop motion techniques to create an animated film. Films may include the use of clay, legos, action figures, armatures, whiteboards, paper cut-outs, etc.
2D
Use traditional or digital hand-drawn animation techniques to create an animation
3D
Animate a scene using a 3D software package

Graphics

Raster
Create an original rasterized illustration or drawing
Vector
Create an original vector illustration, logo, or infographic
3D
Original 3D models and renders created in a 3D modeling program

Photography

Digital
Original student photograph
Composite
Create digital photo manipulations and composites

Interactive Media and Game Design

Interactive Media and Game Design
Original game or interactive experience

Audio and Music

Digital Music Production
Create, record, and edit an original musical composition
Digital Audio Production
Create, record, and edit an original audio production. These may include podcasts, public service announcements, commercials, etc.

Experimental

Experimental Media
Create an original piece of media that transcends the other categories. These may include demo-reels, motion graphics, mixed media, etc.
